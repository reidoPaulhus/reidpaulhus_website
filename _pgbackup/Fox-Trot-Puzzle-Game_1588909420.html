Fox Trot Puzzle Game

A First Person Real-Time & Turn Based Puzzler

The Untitled Puzzle Game is a small side project in which the player must find and free the zoo animals from each room in the science facility by guiding each into a make-shift air duct. The game is considered a first person puzzle game with turn-based elements that allow the player to think and strategize solutions without a hurry.

A culmination of various Unreal Engine tricks I have learned over the years, the Untitled puzzle game is a project that I could only describe as a creative outlet. The conception of the project was due to the ongoing work on other projects, and the limitations that came along with them that prevented me from trying some other ideas such as the implementation of A* for path finding, and practice with material and shader development. I too also use this project to focus on proper PBR lighting techniques to improve the lighting quality.

Work I’m most proud of:

  • Cellular automaton system
  • A* path finding in Blueprints, and then to C++
